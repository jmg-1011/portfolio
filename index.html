안녕하세요!
항상 호기심을 갖고 배우며 성장하는 웹 퍼블리셔 조민근입니다.

웹 퍼블리싱을 처음 접하고 나서, 코딩의 매력에 빠져 배움의 길을 시작했습니다.
배우면 배울수록 더 깊이 알고 싶어지는 이 분야에서, 끊임없이 성장하고자 하는 열정을 가지고 있습니다.

작은 코드 한 줄이 만들어내는 변화와 그로 인해 사용자가 편리함을 느끼는 순간은 저에게 큰 동기부여가 됩니다.
이를 위해 실력을 다지며 다양한 기술을 익히는 데 힘써왔습니다.

그 과정에서 GTQ 1급, 웹 디자인 기능사와 같은 관련 자격증을 취득하며 디자인과 퍼블리싱 역량을 높였습니다.
앞으로도 지속적으로 배우고 발전하는 퍼블리셔가 되어 웹에 가치를 더하겠습니다.

Skill

Html

jQuery

Css

다양한 최신 디자인 트렌드를 반영하고, JavaScript와의 연동을 통해 생동감 있는 애니메이션과 반응형 디자인을 구현할 수 있습니다.

Javascript

슬라이드, 메뉴, 스크롤 애니메이션 등 다양한 동적 기능을 자바스크립트로 구현하여 상호작용이 뛰어난 웹 경험을 제공합니다.

Figma
